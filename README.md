# Brick Breaker

A simple 2D game written in C using SDL2.
Version 1.0 (initial stable release).

## Features

- Basic gameplay loop
- Ball movement
- Paddle movement
- Collision system
- Win/Lose system

## Requirements
- SDL2
- SDL2_ttf

### Install requirements

### Ubuntu / Debian

```
sudo apt install libsdl2-dev libsdl2-ttf-dev
```
Installation tested on Debian 12

## How to run

### Build

```
make
```
### Run

```
make run
```

Or:

```
./build/game
```

### Clean and rebuild

```
make re
```
